# Milk warming calculator

## Deployment

- App should be a self-contained artifact
- Add Docker Compose file to make it easy to migrate and spin up on a home server

## Features

- App will be used to calculate how long and at what temperature to submerge a bottle of breastmilk, using fundamental physics for the calculation, to get it to correct temperature for feeding (body temperature) from being stored in a fridge.

The user input to the calculator will be the quantity of breastmilk being warmed.

Outputs will be:

1. How much hot water (in a Pyrex jug or similar) to submerge the bottle in
2. What temperature to have that water at, at the beginning of the submersion (it will lose temperature naturally during the warming period)
3. How long to submerge the bottle in the Pyrex container with water in it
