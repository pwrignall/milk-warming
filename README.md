# Milk Warming Calculator

A single-page app that uses physics-based heat transfer to calculate how long to submerge a bottle of breastmilk in hot water to bring it from fridge temperature to body temperature.

## Inputs

- Volume of milk in the bottle (ml)

## Outputs

- How much hot water to use
- Starting water temperature
- How long to submerge the bottle

## Running

Open `index.html` directly in a browser, or serve it with Docker Compose:

```bash
docker compose up
```
